# 🌐 Meus Links

Este projeto é uma página pessoal criada para reunir em um só lugar meus principais links:  
**GitHub, LinkedIn, Portfólio e Instagram**.  
A ideia é ter um espaço organizado, moderno e fácil de acessar.

---

## 🚀 Tecnologias utilizadas
- **HTML5** → estrutura da página  
- **CSS3** → estilização com gradientes, cores e responsividade  
- **JavaScript** → alternância de tema (dark/light mode)  
- **Ionicons** → ícones das redes sociais  
- **Google Fonts (Inter)** → tipografia moderna e legível  

---

## ✨ Funcionalidades
- Perfil com foto e nome de usuário  
- Botões de acesso rápido para minhas redes e portfólio  
- Alternância entre tema claro e escuro  
- Rodapé com créditos ao curso Rocketseat 🚀  

---

## 📌 Objetivo
Criar uma página simples e prática para compartilhar meus links pessoais, servindo como cartão de visitas digital.
